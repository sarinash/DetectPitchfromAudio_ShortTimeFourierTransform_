### Dependencies:
* pip install numpy
* pip install librosa
* pip install pandas


Know more about librosa=> https://librosa.org/doc/latest/index.html



### How it works:
_ Extract the stft
_ Calculate the spectrum
_ Calculate the max frequency per frame
_ Export data to .csv
